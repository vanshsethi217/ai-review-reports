# AI Code Review Reports

This repository stores HTML security review reports generated by the [AI Security Review Platform](https://vanshsethi217.github.io/ai-review-frontend/).

## Structure

- `reports/` — All generated scan reports, named by timestamp (e.g. `scan_20260701_085009.html`)
- `.github/workflows/` — GitHub Action that automatically triggers AI review on Pull Requests

## How it works

Reports are automatically generated and published here whenever:
- A repository scan is run from the [dashboard](https://vanshsethi217.github.io/ai-review-frontend/)
- A Pull Request is opened in a repository with the AI Security Review GitHub Action installed

Each report includes a security score (A–F), severity breakdown, and detailed findings with suggested fixes.
